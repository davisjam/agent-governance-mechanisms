# Privacy Policy

**self-governance** — the *agent-governance-mechanisms* plugin — collects no data, executes no code, and
makes no network connections. It is guidance on engineering practice, nothing more.

- **No code execution.** The skill is documentation plus a reasoning stance. It reads files you point it
  at and *proposes* changes; it runs nothing on its own, and ships no hooks, commands, or MCP servers.
- **No network connection.** It contacts no server and sends nothing anywhere. All work happens locally in
  your Claude session.
- **No data collected.** No personal data, no telemetry, no usage tracking — there is no account and no
  backend.
- **Acts only on your approval.** Any change to your files is proposed first and applied only when you
  greenlight it.

In short: **no code executed, no network connection, no privacy concerns — just guidance on engineering
practice.**
